# Alacena 6.0 Premium

Rediseño visual inspirado en iOS/macOS con navegación adaptativa, superficies translúcidas, imágenes locales y todos los módulos funcionales de la versión 5.1.

## Despliegue

Conserva las mismas variables de entorno de Netlify y Supabase. Netlify instalará dependencias y ejecutará `npm run build`.
