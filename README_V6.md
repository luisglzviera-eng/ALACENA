# Alacena v6.0

Versión consolidada con rediseño visual, dashboard renovado y lista del súper modular.

## Despliegue

1. Conserva las variables de entorno actuales en Netlify.
2. Sustituye los archivos del repositorio por el contenido de este proyecto.
3. Haz commit en GitHub.
4. Netlify instalará dependencias y ejecutará `npm run build` automáticamente.

No es necesario volver a crear Supabase ni cambiar las variables de entorno.
